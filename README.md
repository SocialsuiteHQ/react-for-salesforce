# react-for-salesforce
React for Salesforce ISVs


## Install

1. Add packages to VSCode: ESLint, Flow Language Support, Babel ES6/ES7
2. `npm i -g flow-bin` and add to PATH
3. `yarn install`

To test setup run:
`npm run build`
`npm start` and open features/search.html
`npm test`
`npm run storybook`
